# Модуль пользователя

Пользовательская конфигурация через nix-maid.

## Структура

- `games/` — Steam, Gamescope, VR (см. `games/README.md`)
- `nix-maid/` — Управление конфигом
- `session/` — Hyprland, привязки клавиш
- `gui/` — Конфиги GUI-приложений

## Ключевые поддиректории

| Путь | Назначение |
|------|------------|
| `nix-maid/apps/` | Конфиги приложений |
| `nix-maid/cli/` | Оболочка и CLI |
| `session/` | Настройка сессии Hyprland |
