# Модуль разработки

Инструменты разработки, языки и утилиты для исследования безопасности.

## Структура

| Директория | Назначение |
|------------|------------|
| `android/` | Android SDK и инструменты |
| `benchmarks/` | Бенчмарки |
| `editor/` | Редакторы кода |
| `elf/` | Анализ ELF |
| `gcc/` | GCC toolchain |
| `gdb/` | Отладчик |
| `git/` | Конфигурация Git |
| `hack/` | Безопасность (19 модулей) |
| `openxr/` | Разработка OpenXR |
| `pkgs/` | Пакеты для разработки |
| `python/` | Экосистема Python |
| `unreal/` | Unreal Engine |

## Ключевые подмодули

### hack/

Инструменты безопасности:

- **Forensics** — Анализ дисков/памяти
- **Pentest** — разведка, фаззинг, пароли, веб, беспроводные сети
- **Reverse engineering** — дизассемблеры, отладчики

### python/

Разработка на Python:

- Линтеры (ruff, pyright)
- Форматтеры (black, isort)
- Инструменты виртуального окружения

## Переключатель

```nix
features.dev.enable = true;  # Включить
```
