# Модуль фич

Флаги функций для условной конфигурации системы.

## Использование

```nix
features = {
  gui.enable = true;      # Включить GUI
  dev.enable = true;      # Разработка
  cli.enable = true;      # Командная строка
  games.enable = true;    # Игры
};
```

## Назначение

Позволяет выборочно включать компоненты системы в зависимости от роли хоста.
